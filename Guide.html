Guide de Mise en Service - Mada POS

Suivez ces étapes pour transformer votre PC en serveur de restaurant.

1. Préparation du Serveur

Lancez le fichier Mada POS.exe. Lors de l'alerte Windows, cliquez sur "Informations complémentaires" puis "Exécuter quand même".

2. Configuration du Réseau

Vérifiez que le PC est sur le Wi-Fi du restaurant. Dans l'administration (5 clics sur le titre), entrez l'adresse IP du PC (ex: 192.168.1.132).

3. Accès Client & Personnel

Scannez le QR Code généré dans le Dashboard ou tapez l'adresse IP suivie de :5000 dans Safari/Chrome sur vos tablettes.

4. Gestion des Données

Les photos et la base de données sont sauvegardées automatiquement dans votre dossier %AppData%/Roaming/Mada POS.
